BNM_Data_Warehouse: ETL Process for Data Centralization

The BNM_Data_Warehouse project, developed for the **Banque Nationale de Mauritanie (BNM)**, focuses on the creation and implementation of an ETL (Extract, Transform, Load) process to centralize and organize data from multiple sources. This ETL pipeline ensures efficient data extraction, transformation for consistency, and loading into a centralized warehouse, providing a robust foundation for further data analysis and visualization.
